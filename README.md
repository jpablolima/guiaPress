## 📦 Como baixar o projeto 

```
   # Clonar o projeto
   $ git clone https://github.com/jpablolima/guiaPress.git

   # Entrar no diretório do projeto
   $ cd guiaPress

   # Instalar dependências
   $ npm install

   # Iniciar Projeto
   $ npm start
```
